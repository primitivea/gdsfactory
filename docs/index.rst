WhatsApp Web Versi Lama dan Kekurangannya
###########################################

WhatsApp Web merupakan sebuah layanan online yang memungkinkan pengguna untuk melakukan aktivitas WhatsApp dengan menggunakan desktop atau laptop. Pengguna bisa menerima pesan dan mengirimkan pesan secara langsung melalui web browser di komputer, laptop serta desktop, setelah membuka aplikasi WhatsApp yang terpasang di ponsel pengguna. 

Hal tersebut digunakan karena pada prinsipnya WhatsApp Web menggunakan sistem mirroring dari aplikasi WhatsApp yang terpasang pada ponsel pengguna. Perlu diketahui bahwa WhatsApp adalah sebuah aplikasi mobile, sehingga hanya bisa digunakan saat ponsel terhubung dengan jaringan internet.

Sesuai dengan namanya  WhatsApp Web versi lama hadir dalam bentuk web yang bisa dibuka melalui browser internet komputer dan laptop. Pengguna bisa menggunakannya melalui Google Chrome, Mozilla Firefox, Microsoft Edge, dan browser lainnya. Semua aktivitas  WhatsApp seperti pesan atau riwayat panggilan terikat oleh nomor telepon pengguna yang disimpan di ponsel, bukan di server WhatsApp. 

Oleh sebab itu, WhatsApp akan mengambil pesan-pesan tersebut dari ponsel pengguna, sehingga jika ponsel pengguna mati atau tidak terhubung ke jaringan internet maka secara otomatis WhatsApp Web tidak bisa digunakan. Selain itu, pengguna juga tidak dapat menerima, mengirimkan pesan atau melihat status pembaharuan melalui WhatsApp Web.
Namun, belakangan ini WhatsApp Web telah melakukan upgrade atau pembaharuan yang memungkinkan pengguna untuk tetap bisa menggunakan WhatsApp Web meski ponsel pengguna dalam keadaan mati atau tidak terhubung ke jaringan internet setelah login di browser. Selain itu, WhatsApp Web juga bisa digunakan lebih dari satu perangkat.

Kekurangan WhatsApp Web Versi Lama
=======================

#1 Baterai Cepat Habis
----------
Penggunaan WhatsApp Web yang bergantung pada koneksi WhatsApp di ponsel, menjadikan baterai pada ponsel cepat habis. Selain itu, tak jarang beberapa ponsel akan menjadi lebih panas jika dihubungkan dengan WhatsApp Web.

#2 Tidak Bisa Digunakan Jika HP/ Ponsel Mati
----------
Penggunaan WhatsApp Web memiliki ketergantungan penuh pada Hp atau ponsel pengguna. Sehingga, jika ponsel pengguna mati maka secara otomatis WhatsApp Web tidak dapat digunakan.

Syarat Penggunaan WA Web Versi Lama
=====================

Dalam penggunaan WhatsApp Web ada beberapa hal yang perlu pengguna perhatikan. Beberapa hal tersebut merupakan syarat yang harus dipenuhi jika pengguna akan menggunakan WhatsApp Web:

1. Pastikan jika kamu adalah pengguna ponsel Android atau iPhone (iOS).
2. Ponsel Android yang mendukung aplikasi WhatsApp adalah Android dengan sistem 4_0_3. Sedangkan untuk IOS dengan sistem operasi minimum iOS 9
3. Selain kedua perangkat di atas, WhatsApp mendukung beberapa tablet kecuali tablet khusus Wifi.
4. Pastikan pengguna sudah memiliki akun WhatsApp yang aktif di ponsel.
5. Pastikan koneksi internet kuat dan stabil
6. Browser yang bisa digunakan untuk WhatsApp Web antara lain: Chrome, Firefox, Opera, Safari dan Edge.
